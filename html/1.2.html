（二）操作系统发展历程

一、单项选择题

1. 提高单机资源利用率的关键技术是 (          )
题目解析
多道程序设计技术是提高单机资源利用率的关键技术。通过在内存中同时存放多个程序，当一个程序因等待I/O而暂停时，CPU可以切换到另一个程序继续执行，从而大大提高了CPU和其他系统资源的利用率。
2. 批处理系统的主要缺点是 (          )
题目解析
批处理系统的主要缺点是无交互能力。用户提交作业后无法与正在运行的程序进行交互，必须等待作业完成后才能看到结果。这种缺乏实时交互的特点使得批处理系统不适合需要人机对话的应用场景。
3. 下列选项中，不属于多道程序设计的基本特征的是 (          )
题目解析
多道程序设计的基本特征包括：制约性（程序间相互制约）、间断性（程序执行走走停停）、共享性（共享系统资源）。顺序性是单道程序的特征，在多道程序环境中，程序的执行是并发的，不再具有顺序性。
4. 操作系统的基本类型主要有 (          )
题目解析
操作系统的基本类型主要有：批处理操作系统、分时操作系统和实时操作系统。这三种类型分别适用于不同的应用场景：批处理系统适合大量计算任务，分时系统适合多用户交互，实时系统适合对时间要求严格的控制应用。
5. 实时操作系统必须在 (          ) 内处理来自外部的事件。
题目解析
实时操作系统必须在被控制对象规定时间内处理来自外部的事件。这个时间限制是由具体的应用需求决定的，不同的实时系统有不同的时间要求，关键是要在规定的截止时间内完成响应。
6. (          ) 不是设计实时操作系统的主要追求目标。
题目解析
实时操作系统的主要追求目标是安全可靠、及时响应和快速处理。资源利用率不是实时系统的主要目标，实时系统更注重的是能否在规定时间内完成任务，而不是追求最高的资源利用率。
7. 下列 (          ) 应用工作最好采用实时操作系统平台。
I. 航空订票    II. 办公自动化    III. 机床控制    IV. AutoCAD
V. 工资管理系统    VI. 股票交易系统
题目解析
实时操作系统适用于对时间要求严格的应用：I. 航空订票（需要快速响应用户查询）、III. 机床控制（需要精确的时间控制）、VI. 股票交易系统（需要实时处理交易）。办公自动化、AutoCAD和工资管理系统对实时性要求不高。
8. 下列关于分时系统的叙述中，错误的是 (          )
题目解析
分时系统的错误说法是A. 分时系统主要用于批处理作业。分时系统是为了支持多用户交互而设计的，主要用于交互式应用，而不是批处理作业。分时系统通过时间片轮转调度，为每个用户提供快速响应。
9. 分时系统的一个重要性能是系统的响应时间，对操作系统的 (          ) 因素进行改进有利于改善系统的响应时间。
题目解析
改善分时系统响应时间的关键因素是代码可重入。可重入代码允许多个进程同时执行同一段代码的不同实例，减少了进程切换的开销，从而改善响应时间。加大时间片会增加响应时间，静态页式管理和非抢占调度都不利于快速响应。
10. 分时系统追求的目标是 (          )
题目解析
分时系统追求的目标是比较快速响应用户。分时系统的设计目标是为多个用户提供良好的交互体验，通过时间片轮转调度确保每个用户都能得到及时的响应，而不是单纯追求高吞吐率或设备利用率。
11. 在分时系统中，时间片一定时，(          ) 响应时间越长。
题目解析
在分时系统中，时间片一定时，用户数越多响应时间越长。这是因为每个用户都需要等待轮到自己的时间片，用户数量增加意味着等待时间增加，从而导致响应时间变长。响应时间 = 用户数 × 时间片长度。
12. 在分时系统中，为使多个进程能够及时与系统交互，最关键的问题是能在短时间内，使所有就绪进程都能运行。当就绪进程数为100时，为保证响应时间不超过2s，此时的时间片最大应为 (          )
题目解析
根据分时系统响应时间计算公式：响应时间 = 就绪进程数 × 时间片长度。要保证响应时间不超过2s，且就绪进程数为100，则时间片最大为：2s ÷ 100 = 0.02s = 20ms。
13. 操作系统有多种类型。允许多个用户以交互的方式使用计算机的操作系统，称为 (          ) ；允许多个用户将若干作业提交给计算机系统集中处理的操作系统，称为 (          ) ；在 (          ) 的控制下，计算机系统能及时处理由过程控制反馈的数据，并及时做出响应；在IBM-PC中，操作系统称为 (          )
题目解析
这是一道填空题的选择题形式。正确答案依次是：分时操作系统（允许多用户交互）、批处理系统（集中处理作业）、实时操作系统（及时响应控制反馈）、微型计算机操作系统（IBM-PC使用的系统类型）。
14. 下列各种系统中，(          ) 可以使多个进程并行执行。
题目解析
只有多处理器系统可以使多个进程真正并行执行。分时系统、批处理系统和实时系统在单处理器上只能实现并发执行（通过时间片轮转），而不是真正的并行执行。并行需要多个CPU同时工作。
15. 下列关于操作系统的叙述中，正确的是 (          )
题目解析
正确的说法是B. 实时操作系统须在规定时间内处理完来自外部的事件。实时系统的核心特征就是必须在截止时间内完成响应。批处理系统不强调响应时间，分时系统强调的是响应时间而不是周转时间或调度时间。
16. 引入多道程序技术的前提条件之一是系统具有 (          )
题目解析
引入多道程序技术的前提条件之一是系统具有中断功能。中断机制使得CPU能够在程序等待I/O时切换到其他程序执行，这是实现多道程序并发执行的基础。没有中断功能，就无法实现程序间的切换。
17. 【2016统考真题】 下列关于批处理系统的叙述中，正确的是 (          )
I. 批处理系统允许多个用户与计算机直接交互
II. 批处理系统分为单道批处理系统和多道批处理系统
III. 中断技术使得多道批处理系统的I/O设备可与CPU并行工作
题目解析
正确的说法是A. 仅II、III。II正确：批处理系统确实分为单道和多道批处理系统；III正确：中断技术使得I/O设备可与CPU并行工作；I错误：批处理系统不允许用户与计算机直接交互，这是其主要特点。
18. 【2017统考真题】 与单道程序系统相比，多道程序系统的优点是 (          )
I. CPU利用率高    II. 系统开销小    III. 系统吞吐量大    IV. I/O设备利用率高
题目解析
与单道程序系统相比，多道程序系统的优点是D. 仅I、III、IV：I. CPU利用率高（程序切换减少CPU空闲）、III. 系统吞吐量大（同时处理多个程序）、IV. I/O设备利用率高（I/O与CPU并行工作）。II错误：多道程序系统的系统开销更大。
19. 【2018统考真题】 下列关于多任务操作系统的叙述中，正确的是 (          )
I. 具有并发和并行的特点    II. 需要实现对共享资源的保护
III. 需要运行在多CPU的硬件平台上
题目解析
关于多任务操作系统，正确的说法是C. 仅I、II：I正确：多任务系统具有并发特点，在多CPU系统上还具有并行特点；II正确：需要实现对共享资源的保护以避免冲突；III错误：多任务系统不一定需要多CPU，单CPU也可以通过时间片实现多任务。
20. 【2022统考真题】 下列关于多道程序系统的叙述中，不正确的是 (          )
题目解析
不正确的说法是D. 进程数越多CPU利用率越高。虽然多道程序可以提高CPU利用率，但进程数过多会导致系统开销增大（如进程切换开销），反而可能降低CPU利用率。存在一个最优的进程数量，超过这个数量后利用率会下降。
