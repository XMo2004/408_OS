1.4 操作系统结构、引导、虚拟机

1. 用 (          ) 设计的操作系统结构清晰且便于调试。
题目解析
A. 分层式构架设计的操作系统结构清晰且便于调试。分层式结构将操作系统分为若干层，每层只能调用下层提供的服务，层次关系明确，便于理解和调试。模块化、微内核等结构虽然也有各自优点，但在结构清晰性和调试便利性方面不如分层式。
2. 下列关于分层式结构操作系统的说法中，(          ) 是错误的。
题目解析
错误的说法是C. 具有非常灵活的依赖关系。分层式结构的特点是各层之间只能单向依赖，上层只能调用下层服务，不能反向调用，因此依赖关系是严格的单向关系，而不是灵活的依赖关系。这种严格的层次关系正是分层式结构的核心特征。
3. 下列选项中，(          ) 不属于模块化操作系统的特点。
题目解析
不属于模块化操作系统特点的是B. 内核中的某个功能模块出错不会导致整个系统崩溃，可靠性高。这个说法是错误的。在模块化操作系统中，各模块运行在同一地址空间，某个模块出错可能会影响整个系统。而微内核系统中各服务运行在独立地址空间，才具有更高的可靠性。
4. 相对于微内核系统，(          ) 不属于大内核操作系统的缺点。
题目解析
不属于大内核操作系统缺点的是C. 内核切换太慢。实际上，大内核系统由于所有服务都在内核空间运行，系统调用和模块间通信效率较高，内核切换速度较快。内核切换慢是微内核系统的缺点，因为微内核需要频繁进行用户态和内核态之间的切换以及进程间通信。
5. 下列说法中，(          ) 不适合描述微内核操作系统。
题目解析
不适合描述微内核操作系统的是B. 功能分层设计。微内核操作系统的特点是内核足够小，基于客户/服务器(C/S)模式，策略与机制分离。而功能分层设计是分层式操作系统的特点，不是微内核系统的特征。微内核强调的是最小化内核功能，而不是分层设计。
6. 对于以下五种服务，在采用微内核结构的操作系统中，(          ) 不宜放在微内核中。
I. 进程间通信机制   II. 低级I/O   III. 低级进程管理和调度   IV. 中断和陷入处理   V. 文件系统服务
题目解析
在微内核结构中，不宜放在微内核中的是C. 仅V（文件系统服务）。微内核应该只包含最基本的功能：进程间通信机制、低级I/O、低级进程管理和调度、中断和陷入处理等。而文件系统服务属于高级服务，应该作为用户态服务器运行，不应放在微内核中。
7. 相对于传统操作系统结构，采用微内核结构设计和实现操作系统有诸多好处，下列 (          ) 是微内核结构的特点。
I. 使系统更高效   II. 添加系统服务时不必修改内核   III. 微内核结构没有单一内核稳定   IV. 使系统更可靠
题目解析
微内核结构的特点是C. II、IV：II正确：添加系统服务时不必修改内核，因为服务运行在用户态；IV正确：使系统更可靠，因为服务隔离运行。I错误：微内核由于频繁的进程间通信，效率通常较低；III错误：微内核结构比单一内核更稳定可靠。
8. 下列关于操作系统结构的说法中，正确的是 (          )
I. 当前广泛使用的Windows操作系统采用的是分层式OS结构
II. 模块化的OS结构设计的基本原则是，每一层都仅使用其底层所提供的功能和服务，这样就使系统的调试和验证都变得容易
III. 因为微内核结构能有效支持多处理机运行，所以非常适合于分布式系统环境
IV. 采用微内核结构设计和实现操作系统具有诸多好处，如添加系统服务时，不必修改内核、使系统更高效
题目解析
正确的说法是C. II。II错误：这是分层式OS结构的特点，不是模块化OS结构的特点；I错误：Windows采用的是混合内核结构；III正确：微内核确实适合分布式系统；IV错误：微内核添加服务不需修改内核，但效率通常较低，不是更高效。
9. 下列关于微内核操作系统的描述中，不正确的是 (          )
题目解析
不正确的描述是B. 可提高操作系统的执行效率。微内核操作系统由于需要频繁进行用户态和内核态之间的切换，以及大量的进程间通信，执行效率通常比宏内核系统要低。微内核的优势在于可靠性、可移植性和可扩展性，而不是执行效率。
10. 下列关于操作系统外核(exokernel)的说法中，错误的是 (          )
题目解析
错误的说法是C. 外核负责完成进程调度。外核(exokernel)的设计理念是将硬件资源直接暴露给应用程序，让应用程序直接管理硬件资源。外核只负责资源分配和保护，而进程调度等高级功能由应用程序或库操作系统来完成，不是外核的职责。
11. 对于计算机操作系统引导，描述不正确的是 (          )
题目解析
不正确的描述是C. 引导程序会将硬盘中存储的操作系统全部加载到内存中。引导程序只会加载操作系统的核心部分到内存中，而不是全部操作系统。操作系统的其他部分会根据需要动态加载。如果将整个操作系统都加载到内存，会占用大量内存空间，这是不现实也不必要的。
12. 存放操作系统自举程序的芯片是 (          )
题目解析
存放操作系统自举程序的芯片是C. ROM。ROM(只读存储器)是非易失性存储器，断电后数据不会丢失，因此适合存放系统启动时需要的自举程序(BIOS)。SRAM和DRAM都是易失性存储器，断电后数据丢失；CMOS主要用于存储系统配置信息，不是存放自举程序的地方。
13. 计算机操作系统的引导程序位于 (          ) 中。
题目解析
计算机操作系统的引导程序位于D. 硬盘中。虽然BIOS存储在主板ROM中，但操作系统的引导程序(如GRUB、NTLDR等)通常存储在硬盘的引导扇区中。BIOS负责硬件初始化和自检，然后将控制权转交给硬盘中的操作系统引导程序，由引导程序负责加载操作系统。
14. 计算机的启动过程是 (          )
①CPU加电，CS:IP指向FFFFOH；   ②进行操作系统引导；
③执行JMP指令跳转到BIOS；   ④登记BIOS中断例程入口地址；
⑤硬件自检。
题目解析
计算机的启动过程是C. ①③④⑤②。正确的启动顺序是：①CPU加电，CS:IP指向FFFFOH；③执行JMP指令跳转到BIOS；④登记BIOS中断例程入口地址；⑤硬件自检；②进行操作系统引导。这个顺序反映了从硬件初始化到软件加载的完整过程。
15. 检查分区表是否正确，确定哪个分区为活动分区，并在程序结束时将该分区的启动程序(操作系统引导扇区)调入内存加以执行，这是 (          ) 的任务。
题目解析
这是A. MBR的任务。MBR(主引导记录)位于硬盘的第一个扇区，包含分区表和主引导程序。MBR的主要任务就是检查分区表，确定活动分区，然后将活动分区的引导扇区加载到内存中执行。这是操作系统启动过程中的关键步骤。
16. 下列关于虚拟机的说法中，正确的是 (          )
I. 虚拟机可以用软件实现
II. 虚拟机可以用硬件实现
III. 多台虚拟机可同时运行在同一物理机器上，它实现了真正的并行
题目解析
正确的说法是A. I和II：I正确：虚拟机可以用软件实现(如VMware、VirtualBox)；II正确：虚拟机也可以用硬件实现(如Intel VT-x、AMD-V)。III错误：在单CPU系统上，多台虚拟机是通过时间片轮转实现的并发，而不是真正的并行；只有在多CPU系统上才能实现真正的并行。
17. 下列关于VMware workstation虚拟机的说法中，错误的是 (          )
题目解析
错误的说法是B. 虚拟机中只能安装一种操作系统。这是错误的，VMware workstation等虚拟机软件支持在同一台物理机上创建多个虚拟机，每个虚拟机可以安装不同的操作系统，如Windows、Linux、macOS等。这正是虚拟机技术的重要优势之一。
18. 虚拟机的实现离不开虚拟机管理程序 (VMM) ，下列关于VMM的说法中正确的是(          )
I. 第一类VMM直接运行在硬件上，其效率通常高于第二类VMM
II. 由于VMM的上层需要支持操作系统的运行、应用程序的运行，因此实现VMM的代码量通常大于实现一个完整操作系统的代码量
III. VMM可将一台物理机器虚拟化为多台虚拟机器
IV. 为了支持客户操作系统的运行，第二类VMM需要完全运行在最高特权级
题目解析
正确的说法是B. I和III：I正确：第一类VMM直接运行在硬件上，效率高于运行在宿主操作系统上的第二类VMM；III正确：VMM的核心功能就是将物理机虚拟化为多台虚拟机。II错误：VMM代码量通常小于完整操作系统；IV错误：第二类VMM运行在宿主操作系统上，不需要完全运行在最高特权级。
19. 【2013统考真题】计算机开机后，操作系统最终被加载到 (          )
题目解析
操作系统最终被加载到D. RAM。计算机开机后，BIOS首先运行进行硬件自检，然后将操作系统从硬盘加载到内存(RAM)中执行。RAM是易失性存储器，读写速度快，适合运行程序。BIOS、ROM、EPROM都是非易失性存储器，用于存储固化程序，不是操作系统的运行环境。
20. 【2022统考真题】下列选项中，需要在操作系统进行初始化过程中创建的是 (          )
题目解析
需要在操作系统初始化过程中创建的是A. 中断向量表。中断向量表是操作系统管理中断的重要数据结构，需要在系统初始化时建立。文件系统的根目录和索引结点表是文件系统的组成部分，在文件系统挂载时创建；硬盘分区表是在硬盘分区时创建的，不是操作系统初始化时创建。
21. 【2023统考真题】与宏内核操作系统相比，下列特征中，微内核操作系统具有的是 (          )
I. 较好的性能   II. 较高的可靠性   III. 较高的安全性   IV. 较强的可扩展性
题目解析
微内核操作系统具有的特征是D. 仅II、III、IV：II正确：较高的可靠性，因为服务隔离运行；III正确：较高的安全性，因为权限分离；IV正确：较强的可扩展性，因为易于添加新服务。I错误：微内核由于频繁的进程间通信和上下文切换，性能通常不如宏内核。
